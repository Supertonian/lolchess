## 롤토체스 헬퍼

롤토체스 게임을 하면서 전략을 도와주는 앱 React로 만들기

![메인화면](screenshots/mainlayout-v0.1.0.PNG)

### 데모

https://zerostrength.github.io/lolchess/

### 기능

- 챔피언 클릭으로 선택/해제 가능
- 선택된 챔피언 목록에서 레벨 ▲ ▼ 가능
